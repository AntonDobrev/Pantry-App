# Pantry-App
This is sample hybrid app built with Telerik AppBuilder using Ionic and Telerik Backend Services.

## How to run the sample in Telerik AppBuilder:

* Create a new Hybrid Project and select **Clone** then enter the address of this repository.
* Download the repository as a ZIP file and import it in your AppBuilder client.

## How to build it yourself

Follow the two blog posts from here:
* [Ionic and Telerik Backend Services: A Healthy Partnership](http://developer.telerik.com/featured/ionic-telerik-backend-services-healthy-partnership/)
* [Ionic and Telerik Backend Services: A Healthy Partnership Part 2](http://developer.telerik.com/featured/ionic-telerik-backend-services-healthy-partnership-part-2/)
